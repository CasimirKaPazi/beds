Minetest mod "Beds"
===================
by BlockMen (c) 2014

Version: 1.1

About
~~~~~
This mod adds a bed to Minetest which allows to skip the night. To sleep rightclick the bed.

Craft:
wool wool wool
wood wood wood

Notice: You can use any color of wood or wool, mixing different is also possible.
